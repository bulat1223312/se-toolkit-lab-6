# План выполнения задачи 1

## Выбор LLM
Я использую Qwen Code API, развёрнутый на моей ВМ через qwen-code-oai-proxy. Это обеспечивает бесплатный доступ (1000 запросов в день) и полную совместимость с OpenAI API.

## Архитектура агента
- Загрузка переменных окружения из `.env.agent.secret` с помощью `python-dotenv`.
- Получение вопроса из `sys.argv[1]`.
- Создание клиента OpenAI с базовым URL и ключом.
- Отправка запроса в `chat.completions.create` с минимальным системным промптом.
- Извлечение ответа и формирование JSON с полями `answer` и `tool_calls` (пока пустой массив).
- Вывод только JSON в `stdout`, все логи и ошибки — в `stderr`.
- Обработка исключений и выход с ненулевым кодом при ошибке.

## Тестирование
Один регрессионный тест: запуск `agent.py` с тестовым вопросом и проверка, что stdout содержит валидный JSON с обязательными полями.
